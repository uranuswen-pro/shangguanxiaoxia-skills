# ScholarGraph

ScholarGraph is an intelligent academic research assistant that helps you:

- **Discover relevant papers** through semantic literature search
- **Extract key insights** from full-text PDFs and abstracts
- **Map knowledge relationships** via dynamic knowledge graphs
- **Track research trends** with automated progress monitoring
- **Identify gaps** in existing literature using concept analysis

## Components

- `literature-search`: Finds papers across arXiv, PubMed, and Semantic Scholar
- `paper-analyzer`: Extracts claims, methods, and citations from PDFs
- `knowledge-graph`: Builds interactive networks of concepts and papers
- `progress-tracker`: Monitors new publications in your field
- `concept-learner`: Identifies emerging topics and research gaps

## Requirements

- Python 3.9+
- Node.js 18+
- API keys for Semantic Scholar/arXiv (optional but recommended)
