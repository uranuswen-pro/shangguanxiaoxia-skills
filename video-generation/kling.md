# Kling AI

**Best for:** Human motion, longer clips (up to 10s), realistic faces

**API:** https://klingai.com/global/dev/

## Setup

1. Create account: https://app.klingai.com/
2. Get API credentials from developer portal

## Models

- `Kling-V2-Master` — Best quality
- `Kling-V2-1` — Balanced
- `Kling-V2-5-Turbo` — Fast

## Features

- Up to 10 second clips
- Character consistency across frames
- Professional/Standard quality tiers

## Pricing

- ~$1.00 per 10-second Pro video via API
- Consumer plans: 66 credits/day free tier

## Free Tier

- 66 credits daily (resets at midnight UTC)
- Standard quality only
